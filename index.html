CSS

  • a margin indicates the space around an element block

  • an * can act as a wild card or default setting on a style sheet, this can be overridden and changed for diffrent elements
  • the refrence or (selector) is called before the curly brackets such as h1 h2 styles or (declerations) live in the curly brackets
  • classes allow you to style several elements at once. a period precceding card in this case
  • classes are called in the html code with class="refrence" defined in the stylesheet

Git

  • git status: checks what branch we are currently on

  • git pull origin main pulls down current main branch
  • git push pushes changes stored locally into the cloud based git repo
  • git add adds modifications in the current working branch to the staging area -A represents all
  • git commit creates a snapshot of the changes added to the stageing area -M"" allows for messages to be attatched
